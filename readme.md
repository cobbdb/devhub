# CMG DevHub POC

## Setup
```
$ npm i -g grunt-cli
$ npm i
```

## Demo
1. Navigate to www.dcobb.media/cmg/devhub/client
1. Run `$ grunt watch`
1. In a text editor, open `src/source-file-a.js`.
1. Make some changes and save the file.
1. Watch your git username appear on the devhub webpage.
